# Vehicle Trajectory State Estimation Using Nonlinear Filters

This project compares several nonlinear filtering techniques for estimating the trajectory of a moving vehicle using noisy sensor measurements.

## Implemented Filters
- Extended Kalman Filter (EKF)
- Iterative Extended Kalman Filter (IEKF)
- Unscented Kalman Filter (UKF)
- Ensemble Kalman Filter (EnKF)
- Particle Filter (PF)

## Project Overview
The project estimates the trajectory of a moving vehicle using multiple filtering approaches and compares their performance under measurement uncertainty.

## Evaluation
The performance of each filter is evaluated using Root Mean Square Error (RMSE).

## Course Context
This project was developed as part of the Master's course  
**Recursive State Estimation for Dynamic Systems**  
at **Leibniz University Hannover**.

## Repository Structure

- `data/` — input datasets
- `filters/` — filter implementations
- `functions/` — helper and model functions
- `results/` — generated results and plots
- `report/` — presentation/report file
